# 四种 feel-good 激素对比

> **触发背景**：这是对 `raw/journal/2026-04-18.md` ~ `2026-04-20.md` 的 TODO「多巴胺 vs 内啡肽 vs 血清素 vs 催产素」的直接回应。

## 一表对比

| 维度 | [[dopamine]] | [[serotonin]] | [[endorphins]] | [[oxytocin]] |
|------|--------------|---------------|----------------|--------------|
| **核心隐喻** | 奖赏 / 强化 | 情绪平稳 / satisfaction | 内源性吗啡 (endogenous morphine) | 键合 / 触觉 (bonding/cuddle) |
| **产地** | 脑底神经元 | 脑干中心 | 下丘脑 + 垂体 | 下丘脑 → 垂体释放入血 |
| **合成原料** | tyrosine → L-dopa → dopamine | tryptophan（经 BBB → serotonin） | peptide 激素（约 20 种） | peptide 激素 |
| **主功能** | 奖赏、动机、运动控制、强化学习 | 情绪、记忆、睡眠、消化、恐惧应激 | 疼痛 / 压力应答（内源止痛） | 分娩、泌乳、亲子/伴侣键合 |
| **抑郁关联** | 动机 / 专注力下降（副线） | **主因**；SSRI / SNRI 靶点 | — | Postpartum depression 低相关 |
| **疾病典型** | Parkinson's（dopamine 神经元退化） | 抑郁症 | 慢性疼痛管理的替代机制 | 产后抑郁 |
| **运动响应** | 间接（冥想路径） | ↑（tryptophan 释放；runner's high 次因） | ↑（中等强度有氧最佳；runner's high **主因** β-endorphin） | ↑（高强度武术训练后唾液检出） |
| **饮食路径** | 鸡 / 乳 / 牛油果 / 香蕉 / 种子 / 豆（含 tyrosine） | **反直觉**：复杂碳水 + 少量蛋白优于纯蛋白（见下方注释） | — | — |
| **其他激发器** | 目标 / 奖励设计；冥想 | 阳光 / 光疗；冥想 | 针灸；大笑；音乐；紫外线 | 拥抱 / 按摩 / 触觉；群唱 |
| **补剂风险** | L-dopa 需处方（Parkinson 治疗） | 5-HTP 曾关联 EMS 罕见病 | — | 鼻喷 / 口服已被试，效果差（BBB 障碍） |
| **反馈类型** | 负反馈（标准）+ 强化学习特征 | 负反馈 | 负反馈 | **正反馈回路**（哺乳 → 更多 oxytocin） |
| **双重身份** | hormone + neurotransmitter | hormone + neurotransmitter | peptide hormone + neurotransmitter | hormone + neurotransmitter |

**注：serotonin 饮食反直觉** — 纯蛋白 → 其他氨基酸与 tryptophan 竞争 blood-brain barrier → tryptophan 被挤出。复杂碳水 → insulin → 肌肉吸收其他 aa → tryptophan 独占 BBB 通道。

## 场景 → 激素速查

| 你想要的 | 最高效杠杆 |
|----------|-----------|
| 持续动力 / 专注 | **Dopamine**：设计小奖励循环 + 冥想 + tyrosine 饮食 |
| 情绪平稳、抗抑郁 | **Serotonin**：运动 + 阳光 + 复杂碳水 / 轻蛋白搭配 |
| 止痛 / 放松 / 高效恢复 | **Endorphins**：中等强度有氧（30+ min）/ 针灸 / 大笑 / 音乐 |
| 亲密 / 社交温度 / 信任感 | **Oxytocin**：拥抱 / 按摩 / 群唱 / 陪伴眼神 |

## "四合一"最高杠杆：运动

参见 [[exercise-as-hormone-trigger]]——中等-高强度有氧几乎同时提升四种，是单动作多收益的典型。

## 常见误解修正

- ❌ "dopamine = 快乐" → ✅ dopamine = 奖赏/强化/动机；快乐由 serotonin 主导
- ❌ "endorphins 只管运动后 high" → ✅ 大笑、音乐、针灸、性、阳光都能释放
- ❌ "oxytocin 只和生育有关" → ✅ 信任、社交、日常拥抱都触发；正反馈机制
- ❌ "吃蛋白质 = 补 serotonin" → ✅ 反了，纯蛋白反而降；碳水协作才有效
- ❌ "补激素比改变生活方式更强" → ✅ BBB 障碍让多种补剂效果远低于预期（oxytocin / 5-HTP）

## 与 wiki 核心原则的联系

- [[compound-interest-tool]]：运动是典型"低成本 × 四倍收益"——同一动作同时触发四种激素调节，符合"日拱一卒"的高杠杆干预定义
- [[latent-vs-deterministic]]：激素与情绪是典型潜空间（阈值、剂量、个体差异难以精确量化），而饮食含量、运动时长是确定性参数——实践中两层常混用
